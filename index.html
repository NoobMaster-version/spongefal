Projects

Warehouse Navigation System (KDAIL)

Developed an autonomous navigation system for forklifts using SLAM and Ultrawide-band sensors to enable precise indoor positioning within warehouses. The system allows forklifts to navigate autonomously and perform pick-and-place operations with high precision.

SLAM UWB Sensors Autonomous Navigation

Drone Warehouse Navigation (Eyantra)

Competed in IIT Bombay's Eyantra competition, developing a drone system capable of autonomous warehouse navigation using ArUco markers for positioning and custom path-finding algorithms. Advanced to the semi-final level of this prestigious competition.

Drones ArUco Markers Path-finding

Voice Video Manipulator (VVM)

My first ROS2 project that integrated voice commands, computer vision, and robotic manipulation. Created a simulated manipulator in Gazebo that could detect objects, calculate their distance, and perform pick-and-place operations based on voice commands.

ROS2 Gazebo ML Object Detection

Coin Sorting Robot

Developed during a 24-hour hackathon, this project utilized machine learning for coin denomination detection (₹1, ₹2, ₹5, ₹10) and custom hardware for physical sorting. The system ran successfully on a Raspberry Pi 4B.

ML Raspberry Pi Hardware Design

Kurma Quadruped Robot

Built a turtle-inspired quadruped robot for the international Mass Robotics competition. Successfully implemented and refined the gait mechanism at the SRA Lab, advancing to further levels in the competition.

Quadruped Design Gait Mechanism

River Cleaning Robot

Created for an Environmental Studies course in third semester, this boat-based robot was designed to collect trash from rivers. Developed the chassis and hardware components from scratch, with full remote control capability.

Environmental Engineering Chassis Design Remote Control

Self-Balancing Robot

Engineered a Wall-E inspired robot capable of balancing itself on two wheels, implementing control systems to maintain stability while in motion.

Control Systems Balance Algorithms

Gesture Controlled Car

One of my first projects that marked the beginning of my robotics journey. Built an RC car controlled by hand gestures using an MPU6050 sensor. Implemented the ESP-NOW protocol for wireless communication and presented this project in my department with full documentation.

MPU6050 ESP-NOW RC Design

Line Following Maze Solver (SAC)

Competed in the SRA Lab competition with a custom-built line-following robot capable of solving complex mazes. Secured 4th place with efficient pathfinding algorithms and reliable sensor integration.

Line Following Maze Solving Arduino
