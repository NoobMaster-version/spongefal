Warehouse Navigation System (KDAIL)

Project Overview

This project involves developing an autonomous navigation system for forklifts using SLAM (Simultaneous Localization and Mapping) and Ultrawide-band (UWB) sensors. The goal is to enable precise indoor positioning within warehouses, allowing forklifts to navigate autonomously and perform pick-and-place operations with high accuracy.

Technical Details

- Implemented SLAM algorithms for mapping and localization.
- Integrated UWB sensors for enhanced positioning accuracy.
- Developed path planning and obstacle avoidance modules.
- Tested in simulated warehouse environments.

Challenges Faced

One of the biggest challenges was achieving centimeter-level accuracy in dynamic warehouse settings. Handling real-time obstacle avoidance while optimizing for computational efficiency was also a significant hurdle.
