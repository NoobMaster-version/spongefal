Line Following Maze Solver

Project Overview

Built for the SRA Lab competition, this intelligent robot combines line-following capabilities with advanced maze-solving algorithms. The robot can navigate complex mazes by following lines while making smart decisions at intersections and dead ends.

Competition Challenge

The SRA Lab maze competition required robots to:

  • Follow black lines accurately on white surfaces
  • Navigate complex maze patterns with multiple paths
  • Make intelligent decisions at intersections
  • Handle dead ends and backtrack efficiently
  • Find the shortest path to the maze exit
  • Complete the course in minimum time

Technical Design

  • Line Detection: Array of IR sensors for precise line following
  • Maze Mapping: Algorithm to remember visited paths and intersections
  • Decision Making: Logic for choosing paths at intersections
  • Pathfinding: Implementation of maze-solving algorithms like wall-following
  • Motor Control: Precise speed and direction control for smooth navigation

Built With

ESP-IDF IR Sensors Motor Drivers Pathfinding Algorithms Real-time Control Embedded Programming

Maze Solving Algorithm

The robot uses a combination of algorithms for efficient maze solving:

  • Left-Hand Rule: Always prefer left turns when multiple options are available
  • Dead End Detection: Recognize and backtrack from dead ends efficiently
  • Path Memory: Remember visited paths to avoid repeated exploration
  • Intersection Mapping: Track all intersection points and their connections
  • Optimization: Find shortest path after initial maze exploration

Line Following System

  • Sensor Array: Multiple IR sensors for accurate line detection
  • PID Control: Smooth and stable line following with minimal oscillation
  • Speed Control: Dynamic speed adjustment based on line curvature
  • Junction Detection: Identify T-junctions, crossroads, and dead ends
  • Error Correction: Quick recovery when robot loses the line

Engineering Challenges

  • Sensor Calibration: Fine-tuned IR sensors for consistent line detection under various lighting conditions
  • Junction Recognition: Developed robust algorithms to distinguish between different types of intersections
  • Path Optimization: Implemented efficient data structures to store and process maze information
  • Speed vs Accuracy: Balanced fast movement with reliable line following
  • Memory Management: Optimized code for limited microcontroller memory

Competition Achievement

Secured fourth place in the SRA Lab maze-solving competition, demonstrating excellent pathfinding algorithms and reliable sensor integration. The robot successfully navigated complex maze patterns and showed consistent performance across multiple runs.
